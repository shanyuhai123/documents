title: 虚拟机中安装 Linux
---

## 介绍

很多时候直接使用物理机安装 Linux 并不是一个明智的选择，因为你可能频繁的出错，导致频繁的重装系统，可以利用虚拟机打上快照来步步操作，学习 Linux 的使用。


::: danger 注意事项
1. 虚拟机的桥接网卡需要根据宿主机进行替换，若宿主使用了 WIFI 时，虚拟机也需记得替换对应的桥接网卡。
:::
